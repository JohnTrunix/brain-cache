# HTML :simple-html5:

!!! note

    HTML is a language for describing the structure of a Webpage. It can handle:

    - Headings
    - Paragraphs
    - Lists
    - Images
    - Links
    - etc.

## Basic HTML

```html
<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>Heading</h1>
        <p>Paragraph</p>
    </body>
</html>
```
