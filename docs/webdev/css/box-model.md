# CSS Box Model

![CSS Box Model[^1]](./box-model.png)

```css
div {
    width: 100px;
    height: 100px;
    padding: 10px;
    border: 1px solid black;
    margin: 10px;
}
```

## Margins

```css
div {
    margin: 10px;
}
```

!!! note

    Margin is the space outside the border.

## Padding

```css
div {
    padding: 10px;
}
```

!!! note

    Padding is the space inside the border.

## Borders

```css
div {
    border: 1px solid black;
    border-top: 5px solid black;
    border-radius: 5px;
}
```
